## Getting Started

### Adding Bulma
1. Add dependencies:
```bash
npm install bulma -s
npm install sass -s
npm install node-sass@4.14.1 -s
```
2. Create file scss/styles.scss
3. Add to _app.js import:
```javascript
import '../scss/styles.scss'
```
